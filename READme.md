for SAX Parser:
    put sample.xml to your {Apache Tomcat installation dir}/bin directory
    start application
    go to http://{host}:{port}/sax?element=? (where ? is an element name, ex root)
    
for DOM Parser:
    put sample.xml to your {Apache Tomcat installation dir}/bin directory
    start application
    go to http://{host}:{port}/dom?element=? (where ? is an element name, ex root)
